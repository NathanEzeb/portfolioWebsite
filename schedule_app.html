This is a schedule app for android that I created using Java and the Room framework, which simplifies SQLite. This app allows users to create a school schedule by adding, deleting, updating and viewing terms, courses, and assessments.

Features

  • Add terms, courses, and assessments
  • Delete terms, courses, and assessments
  • Update terms, courses, and assessments
  • View terms, courses, and assessments

Technologies Used

  • Java
  • Room Framework / SQLlite
  • Android Studio
