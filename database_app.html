This is a database app that I created using Java and MySQL. The app allows users to add, delete, and update records in a database. The app also has a search feature that allows users to search for records based on specific criteria. The app is designed to be user-friendly and intuitive, with a simple and clean interface. The app is still a work in progress, but I am constantly working on improving it and adding new features.

Features

  • Add records to the database
  • Delete records from the database
  • Update records in the database
  • Search for records based on specific criteria

Technologies Used

  • JavaFX
  • MySQL
  • Intellij IDE
